üöÄ Piper TTS

Piper is a fast, local neural TTS system. Perfect for privacy and offline use.

You can download voices and executables freely.

Explore Piper TTS

Step 1: Install Piper TTS

Download Piper

Download the latest release from the Piper releases page. You'll need the executable for your OS (e.g., `piper.exe` for Windows, `piper` for Linux/macOS).

Place the executable in a convenient location, e.g., `C:\Temp\piper\piper.exe`.

Download Voice Model

Choose a voice model from the Piper releases. For English, a good start is `en_US-lessac-medium.onnx`. Download it and place it in the same directory as the executable.

üí° Custom Voices: Piper supports many languages and voices. Explore them on the official site!

Step 2: Configure OpenClaw for TTS

Edit your OpenClaw config file (`~/.openclaw/openclaw.json`).

Add Piper TTS Configuration

{
  "tools": {
    "media": {
      "audio": {
        "enabled": true,
        "models": [
          {
            "provider": "piper",
            "type": "cli",
            "command": "C:\\Temp\\piper\\piper.exe", 
            "language": "en-US",
            "voice": "en_US-lessac-medium.onnx",
            "output_file": "{output}",
            "espeak_data": "C:\\Temp\\piper\\piper\\espeak-ng-data"
          }
        ]
      }
    }
  }
}
‚ö†Ô∏è Paths: Make sure the `command`, `voice`, and `espeak_data` paths exactly match where you installed Piper and its model.

Step 3: Voice Input (Microphone)

OpenClaw can use your system's default microphone for input. Most messaging platforms (Telegram, Discord bots) will automatically handle voice messages. For direct commands involving voice, OpenClaw uses system defaults.

# Example: Sending a voice message via Telegram
# (This is handled by the platform integration, not direct command)
openclaw message --channel telegram --action send --message "This is a voice message" --asVoice
üí° Node Support: If using OpenClaw on a less common OS or environment, you might need to configure audio input drivers. Check OpenClaw documentation for your specific setup.

Step 4: Test Voice Output

Ask your AI assistant to say something! This tests the TTS setup.

# In OpenClaw chat
/tts Hello, can you hear me?

You should hear the response spoken by your chosen Piper voice.
