Step 1: Media Processing Tools

FFmpeg (Essential)

The gold standard for video/audio processing. Your AI can convert, compress, edit, and analyze media files.

# Windows (via winget)
winget install Gyan.FFmpeg

# macOS
brew install ffmpeg

# Linux (Ubuntu/Debian)
sudo apt update && sudo apt install ffmpeg

# Verify installation
ffmpeg -version
üí° Common Uses: Convert video formats, extract audio from video, compress files for sharing, create thumbnails.

yt-dlp (Media Downloader)

Download videos and audio from YouTube and 1000+ other sites. Perfect for content analysis and archiving.

# Install via pip (works on all platforms)
pip install yt-dlp

# Or on macOS/Linux with brew
brew install yt-dlp

# Download a video
yt-dlp "https://youtube.com/watch?v=..."

# Extract audio only
yt-dlp -x --audio-format mp3 "https://youtube.com/watch?v=..."

Whisper (Speech-to-Text)

OpenAI's Whisper for transcribing audio. Local processing option available.

# Install
pip install openai-whisper

# Basic usage
whisper audio_file.mp3 --model medium

# Supported formats: mp3, wav, m4a, flac, and more

Step 2: Development Tools

GitHub CLI (gh)

Manage repositories, issues, PRs, and workflows directly from the command line.

# Windows
winget install GitHub.cli

# macOS
brew install gh

# Linux
curl -fsSL https://cli.github.com/packages/githubcli-archive-keyring.gpg | sudo dd of=/usr/share/keyrings/githubcli-archive-keyring.gpg
echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/githubcli-archive-keyring.gpg] https://cli.github.com/packages stable main" | sudo tee /etc/apt/sources.list.d/github-cli.list > /dev/null
sudo apt update && sudo apt install gh

# Authenticate
gh auth login

# Clone a repo
gh repo clone owner/repo

Git (Version Control)

You probably already have this, but ensure it's configured:

# Configure git
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

# Create SSH key for GitHub
ssh-keygen -t ed25519 -C "your.email@example.com"
cat ~/.ssh/id_ed25519.pub
