üé¨ Media Processing Tools

1. FFmpeg ‚Äî Video/Audio Swiss Army Knife

Convert, compress, trim, merge, extract audio, create GIFs, generate thumbnails.

# Install
# Windows: Download from ffmpeg.org, add to PATH
# macOS: brew install ffmpeg
# Linux: sudo apt install ffmpeg

# Common uses in OpenClaw:
ffmpeg -i input.mp4 -ss 00:01:00 -t 30 output.mp4  # Trim 30s starting at 1min
ffmpeg -i video.mp4 -vn -acodec copy audio.aac      # Extract audio
ffmpeg -i input.mp4 -vf "fps=10,scale=480:-1" output.gif  # Create GIF

2. yt-dlp ‚Äî Download from YouTube & 1000+ sites

Download videos, extract transcripts, get metadata, thumbnails.

# Install
pip3 install yt-dlp

# Basic download
yt-dlp "https://youtube.com/watch?v=..."

# Extract audio only (for transcription)
yt-dlp -x --audio-format mp3 "https://youtube.com/watch?v=..."

# Get transcript
yt-dlp --write-subs --sub-langs en "https://youtube.com/watch?v=..."

3. ImageMagick ‚Äî Image manipulation

Resize, convert formats, create composites, generate memes, batch process.

# Install
# Windows: Download from imagemagick.org
# macOS: brew install imagemagick
# Linux: sudo apt install imagemagick

# Common operations
convert input.jpg -resize 800x600 output.jpg
convert *.jpg -append vertical.jpg  # Stack vertically
convert -size 400x200 xc:lightblue -pointsize 30 -fill black -gravity center -annotate +0+0 "Hello" output.png

4. Pandoc ‚Äî Document converter

Convert between Markdown, PDF, Word, HTML, EPUB, LaTeX, and 40+ formats.

# Install
# Windows: Download from pandoc.org
# macOS/Linux: Built-in or package manager

# Markdown to PDF
pandoc document.md -o document.pdf

# Web page to Markdown
pandoc https://example.com -o example.md

# Word to Markdown
pandoc document.docx -o document.md

üåê Web & Data Tools

5. curl ‚Äî HTTP requests

Already built into OpenClaw, but know the power: API calls, downloads, webhooks.

# GET request
curl https://api.example.com/data

# POST with JSON
curl -X POST -H "Content-Type: application/json" \
  -d '{"key":"value"}' https://api.example.com/submit

# Download file

6. jq ‚Äî JSON processor

Parse, filter, transform JSON data from APIs.

# Install
# Windows: Download from stedolan.github.io/jq
# macOS: brew install jq
# Linux: sudo apt install jq

# Extract specific field
curl https://api.github.com/users/octocat | jq '.login'

# Filter arrays
curl https://api.example.com/items | jq '.[] | select(.price < 100)'

# Transform structure
cat data.json | jq '{name: .firstName, email: .contact.email}'

7. pup ‚Äî HTML parsing (like jq for HTML)

Scrape websites, extract data from HTML without writing code.

# Install
# macOS: brew install pup
# Others: Download from github.com/ericchiang/pup

# Extract all links
curl https://example.com | pup 'a attr{href}'

# Extract article titles
curl https://news.ycombinator.com | pup 'span.titleline>a text{}'

# Get meta description
curl https://example.com | pup 'meta[name=description] attr{content}'

8. sqlite3 ‚Äî Database

Lightweight SQL database perfect for storing structured data locally.

# Usually pre-installed

# Create database
sqlite3 workspace/data.db "CREATE TABLE logs (id INTEGER PRIMARY KEY, message TEXT, timestamp DATETIME);"

# Insert data
sqlite3 workspace/data.db "INSERT INTO logs (message, timestamp) VALUES ('System started', datetime('now'));"

# Query
sqlite3 workspace/data.db "SELECT * FROM logs ORDER BY timestamp DESC LIMIT 10;"

Integrating with OpenClaw

Create a wrapper script so OpenClaw can call any tool:

# tools_wrapper.py - Let OpenClaw access all tools

import subprocess
import shlex
from typing import List, Tuple

class ToolRunner:
    """Safe wrapper for running system tools"""
    
    ALLOWED_TOOLS = [
        'ffmpeg', 'yt-dlp', 'convert', 'pandoc', 'curl', 'jq',
        'pup', 'sqlite3', 'gh', 'docker', 'htop', 'ncdu',
        'lsof', 'watch', 'openssl', 'nmap', 'ssh', 'fzf',
        'rg', 'fd', 'bat', 'python', 'node', 'git'
    ]
    
    def run(self, command: str, timeout: int = 60) -> Tuple[str, str, int]:
        """
        Run a tool safely
        
        Returns: (stdout, stderr, return_code)
        """
        # Parse command
        parts = shlex.split(command)
        tool = parts[0]
        
        # Security check
        if tool not in self.ALLOWED_TOOLS:
            return "", f"Tool '{tool}' not in allowed list", 1
        
        try:
            result = subprocess.run(
                parts,
                capture_output=True,
                text=True,
                timeout=timeout,
                cwd="workspace"
            )
            return result.stdout, result.stderr, result.returncode
        except subprocess.TimeoutExpired:
            return "", "Command timed out", 1
        except Exception as e:
            return "", str(e), 1

# Usage in OpenClaw
runner = ToolRunner()
stdout, stderr, code = runner.run("yt-dlp --list-formats https://youtube.com/watch?v=...")
Security Note: Always validate commands before execution. Never pass raw user input directly to shell commands. Use the whitelist approach shown above.

import os
from datetime import datetime

def screenshot_desktop():
    """Take a screenshot"""
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    filename = f"workspace/screenshots/capture_{timestamp}.png"
    os.makedirs("workspace/screenshots", exist_ok=True)

        # Use PowerShell
        pass
    else:  # macOS/Linux
        os.system(f"screencapture {filename}")  # macOS
        # or: os.system(f"gnome-screenshot -f {filename}")  # Linux

def generate_password(length=16):
    """Generate secure random password"""
    import secrets
    import string
    alphabet = string.ascii_letters + string.digits + "!@#$%^&*"
    return ''.join(secrets.choice(alphabet) for _ in range(length))

def compress_folder(folder_path, output_name=None):
    """Compress folder to zip"""
    import shutil
    if not output_name:
        output_name = f"{folder_path}.zip"
    shutil.make_archive(output_name.replace('.zip', ''), 'zip', folder_path)
    return output_name
