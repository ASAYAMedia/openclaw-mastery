Step 1: Install OpenClaw

Windows (PowerShell)

# Install OpenClaw CLI
npm install -g openclaw

# Verify installation
openclaw --version

# Initialize your workspace
openclaw init

macOS / Linux

# Install OpenClaw CLI
npm install -g openclaw

# Or with Homebrew
brew install openclaw

# Verify
openclaw --version
openclaw init
üí° Pro Tip: The `init` command creates your workspace at `~/.openclaw/workspace`. This is where all your files, memory, and configurations live.

Step 3: Configure OpenClaw

Edit your config file:

# Windows
notepad %USERPROFILE%\.openclaw\openclaw.json

# macOS/Linux
nano ~/.openclaw/openclaw.json

Add your API key:

{
  "env": {
    "FIREWORKS_API_KEY": "your-key-here"
  },
  "agents": {
    "defaults": {
      "model": {
        "primary": "fireworks/accounts/fireworks/models/kimi-k2p5"
      }
    }
  }
}
‚ö†Ô∏è Security: Never share your API keys or commit them to public repos. OpenClaw stores them locally in your config file.

Step 5: Connect Telegram (Optional)

Want to chat from your phone? Set up Telegram:

  1. Message @BotFather on Telegram
  2. Type /newbot and follow prompts
  3. Copy the bot token (looks like: 123456789:ABCdefGHIjklMNOpqrsTUVwxyz)
  4. Add to your config:
{
  "channels": {
    "telegram": {
      "botToken": "YOUR_BOT_TOKEN_HERE"
    }
  }
}
