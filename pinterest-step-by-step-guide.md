## PART 6: Website Verification

### Step 1: Go to Settings
1. Click your profile picture (top right)
2. Click **"Settings"**
3. Click **"Claimed accounts"** (left sidebar)

### Step 2: Add Website
1. Enter your website URL
2. Click **"Claim"**

### Step 3: Choose Verification Method
**Option A: HTML File (Easiest)**
1. Download the HTML file Pinterest gives you
2. It will be named something like: `pinterest-xxxxxxxx.html`
3. Upload to your website root directory
4. Click **"Verify"**

**Option B: HTML Tag**
1. Copy the meta tag they provide (looks like: `<meta name="p:domain_verify" content="CODE"/>`)
2. Replace the placeholder in your index.html:
   - Find: `content="8e5f2a7e"`
   - Replace with: `content="YOUR_CODE_FROM_PINTEREST"`
3. Save and reupload index.html
4. Click **"Verify"**

### Step 4: Wait for Confirmation
- Should say "Domain claimed" within 1-2 minutes
- If it fails, check the file is accessible at: `yoursite.com/pinterest-xxxxxxxx.html`

## PART 7: Enable Rich Pins

### Step 1: Go to Rich Pins Validator
1. Visit: **richpins.pinterest.com**
2. Or search "Pinterest Rich Pins" in Google

### Step 2: Validate Your Site
1. Enter your website URL
2. Click **"Validate"**
3. Pinterest will check for proper meta tags
4. Should show green checkmarks for:
   - Article markup detected
   - Valid structured data

### Step 3: Apply
1. Click **"Apply"** 
2. Choose: **"Apply for all my pins"**
3. Takes 24-48 hours to activate

## PART 8: Set Up Pinterest Tag (For Tracking)

### Step 1: Get Tag Code
1. Go to: **ads.pinterest.com** (even if not running ads)
2. Click **"Create ad"**
3. Skip to **"Install Pinterest tag"**
4. Copy the base code

### Step 2: Add to Website
1. Add the tag code to your website's `<head>` section
2. Should look like:
```html
<script>
!function(e){...}(window,document);
pintrk('load', 'YOUR_TAG_ID');
pintrk('page');
</script>
```

### Step 3: Verify
1. Use Pinterest Tag Helper Chrome extension
2. Visit your site
3. Should show tag is firing

## PART 10: Check Analytics

### Weekly Review
1. Go to: **analytics.pinterest.com**
2. Check these metrics:

**Key Numbers to Track**
- **Impressions**: How many see your pins (goal: 10K+/month)
- **Engagement rate**: Saves + clicks / impressions (goal: 2%+)
- **Outbound clicks**: Traffic to your site (the money metric)
- **Saves**: Indicates high intent

### Monthly Goals
