Step 1: Telegram (Easiest)

Create Your Bot

  1. Open Telegram, search for @BotFather
  2. Click Start, then type /newbot
  3. Name your bot (e.g., "MyAI Assistant")
  4. Choose a username (must end in "bot", e.g., "myai_claw_bot")
  5. Copy the HTTP API token

Configure OpenClaw

{
  "channels": {
    "telegram": {
      "botToken": "123456789:YOUR_TOKEN_HERE",
      "enabled": true
    }
  }
}
üí° Find your personal chat ID: Message your bot, then visit https://api.telegram.org/botYOUR_TOKEN/getUpdates ‚Äî look for "chat":{"id":12345678

Step 2: WhatsApp (QR Code)

WhatsApp requires linking via QR code scan:

# In OpenClaw
openclaw whatsapp login
  1. Run the command above
  2. A QR code will appear (or a link to view it)
  3. Open WhatsApp on your phone
  4. Settings ‚Üí Linked Devices ‚Üí Link a Device
  5. Scan the QR code
  6. Done! Your AI can now read/send WhatsApp messages
‚ö†Ô∏è WhatsApp Limitation: You can only have one linked device for WhatsApp Web. If you use WhatsApp Web on your computer, you'll need to re-link when switching.

Step 3: Discord Server Bot

Create Discord Application

  1. Go to Discord Developer Portal
  2. Click "New Application", name it
  3. Go to "Bot" tab ‚Üí "Add Bot"
  4. Copy the bot token (click "Reset Token" if needed)
  5. Enable "Message Content Intent" (critical!)

Invite Bot to Server

  1. Go to OAuth2 ‚Üí URL Generator
  2. Scopes: bot, applications.commands
  3. Bot Permissions: Send Messages, Read Messages, Connect, Speak
  4. Copy the generated URL
  5. Open in browser, select your server

Configure OpenClaw

{
  "channels": {
    "discord": {
      "botToken": "YOUR_DISCORD_TOKEN",
      "enabled": true
    }
  }
}
