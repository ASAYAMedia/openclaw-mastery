## Affiliate Link Strategy

### Direct Links (Allowed on Pinterest)
- Link pins directly to Easyship, Gemini, Shopify, MarketXLS using your affiliate URLs
- Use URL shortener (bit.ly) for cleaner look if needed
- Disclose: "#affiliate" or "(ad)" in description

### Indirect Links (Recommended)
- Link to your tutorial page FIRST
- User reads valuable content
- Clicks affiliate link from your site
- Better conversion + compliance

### Rich Pins (Enable These)
