## Usage Tips for Image Generators

### For Gemini / Nano Banana:
- Use detailed prompts with specific colors
- Mention "Pinterest pin design" and "2:3 ratio" explicitly
- Include hex codes for brand colors
- Request "flat design" or "modern minimalist" style

### For Midjourney:
- Add "--ar 2:3" for aspect ratio
- Add "--style raw" for less stylization
- Use "infographic" or "marketing graphic" keywords
- Specify "no photorealistic elements" to avoid photos

### For DALL-E:
- Very explicit about layout and text placement
- Request "text readable" or "clear typography"
- Specify "digital graphic" not "photograph"

### For Stable Diffusion / FLUX:
- Use the long detailed prompts
- Specify "flat vector style" or "minimalist design"
- Include negative prompts: "no photo, no realistic, vector graphic only"
