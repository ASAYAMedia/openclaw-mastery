## Canva Setup Instructions

### Step 1: Create Account
1. Go to canva.com
2. Sign up (free tier works fine)
3. Search "Pinterest Pin" template

### Step 2: Use These Settings
- **Custom size**: 1000 x 1500 px
- **Color palette**: Add #667eea, #764ba2, #ff6b6b
- **Upload fonts**: Montserrat, Open Sans (or use similar)

### Step 3: Download & Upload
- **Format**: PNG (for quality) or JPG (smaller size)
- **Name files clearly**: 
  - pin-01-build-ai-assistant.png
  - pin-02-23-tools.png
  - pin-03-easyship.png

### Step 4: Upload to Pinterest
1. Go to pinterest.com
2. Click "Create" â†’ "Create Pin"
3. Upload image
4. Copy/paste title & description from above
5. Add link
6. Select board
7. Publish!
