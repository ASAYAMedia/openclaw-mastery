Windows Instructions

Method 1: Snipping Tool (Recommended)

  1. Open the pin HTML file in Chrome
  2. Press Ctrl+0 to ensure 100% zoom
  3. Press Windows key + Shift + S
  4. Select "Rectangle Snip"
  5. Drag to select the entire pin (1000x1500px)
  6. Save as PNG

Method 2: Chrome DevTools (Perfect Quality)

  1. Open pin HTML in Chrome
  2. Press F12 (opens DevTools)
  3. Press Ctrl+Shift+P
  4. Type "screenshot" and select "Capture node screenshot"
  5. Or use: document.querySelector('.pin-container') in console
  6. Right-click element ‚Üí Screenshot node

Mac Instructions

Method 1: Screenshot Tool

  1. Open the pin HTML file in Chrome/Safari
  2. Press Cmd+0 for 100% zoom
  3. Press Cmd+Shift+4
  4. Press Space then click the window
  5. Saves to desktop as PNG

Method 2: Perfect Pixel Dimensions

  1. Open pin HTML in Chrome
  2. Right-click ‚Üí Inspect
  3. Click the device toolbar icon (üì±) or Cmd+Shift+M
  4. Set dimensions to 1000 x 1500
  5. Right-click pin ‚Üí Screenshot node

Upload to Pinterest

  1. Go to pinterest.com
  2. Click "Create" ‚Üí "Create Pin"
  3. Drag and drop your screenshot PNG
  4. Copy/paste title from the file below
  5. Copy/paste description (includes hashtags)
  6. Add your link (affiliate or tutorial URL)
  7. Select board
  8. Click "Publish"
